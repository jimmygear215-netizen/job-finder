## 🔥 QUICK COMMANDS:

```bash
# Install dependencies
pip install flask

# Backup the_working → the_backup (before each deploy)
Copy-Item the_working the_backup/v0.0.X.fullproduction.backup -Recurse -Force

# Deploy from the_dev → the_working
Copy-Item the_dev/app.py the_working/app.py -Force
Copy-Item the_dev/templates/index.html the_working/templates/index.html -Force
Copy-Item the_dev/requirements.txt the_working/requirements.txt -Force

# Run the app
cd the_working
python app.py

# Open in browser
http://localhost:5000
```
