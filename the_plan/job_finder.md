# 🎯 JOB FINDER APP PROJECT

## What We're Building:
A **Resume-to-Job-Matcher Web App** that analyzes your experience and directs you to the best job fits! 

## Quick Start Commands (After Files Are Created):

```bash
# Install dependencies
pip install flask pywebview

# Run the app
cd c:\jimdev\job_finder
python app.py

# Open in browser: http://localhost:5000
```

## Tech Stack:
- **Backend:** Flask (Python) - Fast, lightweight API
- **Frontend:** HTML + JavaScript - Clean, responsive UI  
- **Desktop Wrapper:** PyWebView → Electron (later phases)
