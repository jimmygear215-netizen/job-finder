--------------------------------------------------------------------------------

  → Fast, lightweight, easy API endpoints
  → Runs locally on localhost:5000
  
FRONTEND: HTML + Vanilla JavaScript
  → Simple forms, no build step needed
  → Can wrap in PyWebView or Electron later

DESKTOP WRAPPER: PyWebView OR Electron (Later Phase)
  → PyWebView = lighter wrapper for Python apps
  → Electron = cross-platform native feel
